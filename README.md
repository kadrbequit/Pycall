## 🚀 Özellikler

- ✨ **Renkli Terminal Arayüzü** - Göz yormayan renkli tasarım
- 🎯 **Rastgele User-Agent** - Tespit edilmeyi zorlaştırır
- 🔄 **Otomatik Döngü** - Belirlediğiniz sayıda gönderim
- ⏱️ **Özelleştirilebilir Bekleme** - İstekler arası süreyi ayarlayın
- 📊 **Detaylı Rapor** - Başarılı/Başarısız istatistikleri
- 🛡️ **Gelişmiş Güvenlik** - Rastgele parametreler ile izlenmeyi zorlaştırır

## 📦 Kurulum

```bash
# Repoyu klonlayın
git clone https://github.com/kadrbequit/Pycall.git

# Proje dizinine gidin
cd Pycall

# Gerekli kütüphaneleri yükleyin
pip install -r requirements.txt

#kullanım
python Pycall.py 
```
